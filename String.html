The repeat count must be positive and less than infinity.
Returns:
Type
String
Example
"abc".repeat(0)   // ""
"abc".repeat(1)   // "abc"
"abc".repeat(2)   // "abcabc"
"abc".repeat(3.5) // "abcabcabc" (count will be converted to integer)
"0".repeat(5)     // "00000"
