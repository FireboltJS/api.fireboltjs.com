show(style)

Shows an element by giving it a certain display style. If no parameter is passed in, Firebolt determines the element's default display style and sets it to that (NOTE: the element's default display style may be 'none' and hence the element would not be shown).

Parameters:
Name Type Argument Description
style Number | String <optional>

The style of display the element should be shown with. Possibilities are:

  • 0 => 'block'

  • 1 => 'inline-block'

  • 2 => 'inline'
For other display types, only the string parameter will be accepted.
If the arguement is left blank, the element's default style will be used.
