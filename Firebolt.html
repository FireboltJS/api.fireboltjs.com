<static> Firebolt(string) → {NodeList}

Returns a list of the elements either found in the DOM that match the passed in CSS selector or created by passing an HTML string.
When passed a CSS selector string, acts as an alias of document.querySelectorAll().
Also represents the Firebolt namespace object and can be referenced by the synonyms FB and $ (on pages where $ has not already been defined).

NOTE: When a selector is passed into this function that is of the form "#elid" (for selecting a single element by its id), only a single element is returned instead of a NodeList (or null if there are no elements with the id).

Parameters:
Name Type Description
string String

A CSS selector string or an HTML string.
Source:
  • firebolt.js, line 563
Returns:

A non-live list of selected elements or newly created elements.

Type
NodeList
Example
$('button.btn-success') // Returns all button elements with the class "btn-success"
$('str <p>content<\/p>') // Creates a set of nodes and returns it as a NodeList (in this case ["str ", <p>content</p>])
$('1<br>2<br>3 >');     // Returns ["1", <br>​, "2", <br>​, "3 >"]
$.create('div')         // Calls Firebolt's `create()` method to create a new div element

<static> isEmpty(value, allowEmptyString) → {Boolean}

Determines if the passed in value is considered empty. The value is considered empty if it is one of the following:

  • null
  • undefined
  • a zero-length array
  • an empty object (as defined by Firebolt.isEmptyObject)
  • a zero-length string (unless the allowEmptyString parameter is set to a truthy value)
