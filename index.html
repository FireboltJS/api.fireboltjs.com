Firebolt

Firebolt is a fast and lightweight JavaScript library for modern browsers. Its core functionality is based on jQuery, so if you already know how to use jQuery, you know how to use Firebolt (with a few minor exceptions).

WARNING

This project is still in the early stages of development. The API could change at any time and code is assumed to be broken.

Browser Support

  • Internet Explorer 9+
  • Chrome 30+
  • Firefox 24+
  • Safari 5.1+
  • Opera 12.1+
  • iOS 5.1+
  • Android 3.0+
  • Blackberry 10+
  • IE Mobile 10+
